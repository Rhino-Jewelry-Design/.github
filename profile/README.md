# Rhino Software — NURBS 3D Modeling Environment

## About Rhino Software

Rhino Software provides a complete NURBS‑based 3D modeling environment for industrial design, jewelry rendering, CAM preparation, SubD organic surfaces, Grasshopper visual scripting, and mesh repair. Rhino Software combines analytic surfaces (cones, spheres, toruses, ellipsoids, paraboloids), deformations (twist, taper, bend, flow along surface, cage edit), and surface continuity inspection (zebra, curvature analysis, ambient occlusion). Rhino Software supports jewel‑specific plugins (RhinoGold, RhinoJewel, TechGems, Panther, MatrixGold integration) and print‑to‑3D (STL export with manifold check, support generator export) directly into the interface. Rhino Software enables users to model precise curved geometry, fabricate molds, and render photorealistic product visuals. Rhino Software runs on portable workspaces and dedicated industrial design stations. Rhino Software is trusted by industrial designers, jewelers, product engineers, automotive stylists, and CAD/CAM specialists worldwide.

## Deep Dive — NURBS Modeling Notes

NURBS (Non‑Uniform Rational B‑Spline) mathematics: control points (edit location weight), degree (1=linear, 2=quadratic, 3=cubic, 5, 7), knots (uniform, chord length, centripetal), evaluation (point at parameter, derivative, curvature vector). Real‑time continuity switching: surface continuity matching (position G0, tangent G1, curvature G2, curvature change G3).
